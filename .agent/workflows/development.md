## 開発規約

すべての開発規約は `CONTRIBUTING.md` に記載されています。

開発を行う際は、以下を参照してください：
- ブランチ命名規則
- コミットメッセージ規約
- Pull Request規約
- コーディング規約

## CIチェック

PRを作成すると、以下が自動でチェックされます：
- Lint (`npm run lint`)
- Build (`npm run build`)
- Test (`npm run test`)
- PRタイトルとブランチ名の形式
